# BeGood

Мы разработали приложение, которое помогает связывать волонтёров и организаторов мероприятий. Главная концепция нашего проекта — это простота и удобство использования, минимизация лишних действий и максимальная эффективность. Наше приложение под названием "BeGOOD" призвано вдохновлять людей на добрые поступки.
